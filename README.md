# MerossIPSymcon

IP-Symcon Modul zur lokalen Steuerung der Meross MSS210 / MSS210p Steckdose.

## Installation in IP-Symcon

1. Kern-Instanzen → Module → + → URL eintragen:
   https://github.com/DEIN-NAME/MerossIPSymcon

2. Objektbaum → Rechtsklick → Instanz hinzufügen → "Meross" suchen → MerossControl
